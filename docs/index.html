Core Features

为 AI 驱动的 SRE 运维而设计

🔧

36 个 MCP 工具

覆盖 TKE、CVM、CLB、CDB、VPC、Region 六大产品线，提供全面的只读查询能力

🌐

多传输模式

支持 stdio / HTTP / SSE 三种通信协议，灵活适配不同集成场景

🔒

安全认证

Bearer Token / Basic Auth / API Key + IP 白名单，多重安全保障

☸

Kubernetes 公共库

独立 Go 模块，支持命名空间、工作负载、Pod 状态的标准化查询

📊

双输出格式

所有云产品工具同时支持 JSON 和 Table 两种输出格式

🚀

内网加速

支持通过内网域名访问腾讯云 API，部署在云上时延迟更低、无需公网带宽

Quick Start

几分钟内启动你的 AI SRE 工具服务

Terminal - Build & Run 
# Clone the repository
git clone https://github.com/goops-top/ai-sre.git
cd ai-sre/tools/mcp

# Build
go build -o bin/mcp-server ./cmd/mcp-server

# Run in HTTP mode
export TENCENTCLOUD_SECRET_ID="your_secret_id"
export TENCENTCLOUD_SECRET_KEY="your_secret_key"
./bin/mcp-server -transport http -port 8080
Terminal - Docker Deploy 
# Docker one-liner
docker run -d \
  --name ai-sre-mcp-server \
  -p 8080:8080 \
  -e TENCENTCLOUD_SECRET_ID="your_secret_id" \
  -e TENCENTCLOUD_SECRET_KEY="your_secret_key" \
  -e TENCENTCLOUD_REGION="ap-beijing" \
  ai-sre-mcp-server:latest
